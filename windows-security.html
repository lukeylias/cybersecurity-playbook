Windows Security

Built-in Protection & Defence Tools

Windows ships with layered security ‚Äî software-level protection (Defender, Firewall, SmartScreen), hardware-level defences (TPM, Core Isolation), and data protection (BitLocker, Volume Shadow Copy). All managed through Windows Security.

Windows Security uses a traffic light system across all sections ‚Äî green checkmark means protected, yellow triangle means a recommendation to review, red X means immediate attention is needed.

üõ°Ô∏è Virus & Threat Protection
Managed by Microsoft Defender, this handles malware scanning and real-time threat prevention.
Scan Types
Quick scan
Checks common locations where threats are typically found. Fast but not comprehensive.
Full scan
Scans every file and program on the system. Thorough but takes a long time.
Custom scan
You choose exactly which files or folders to scan.

Threat history shows the last scan time, quarantined threats (isolated so they can't run), and allowed threats (items you've manually permitted).

Protection Settings
Real-time protection
Actively monitors and blocks malware as it happens. This is the most critical setting ‚Äî keep it on.
Cloud-delivered protection
Sends suspicious files to Microsoft's cloud for faster identification of new threats.
Automatic sample submission
Automatically sends file samples to Microsoft to help improve detection.
Controlled folder access
Locks down specified folders so only trusted apps can modify files inside them. Also required for ransomware protection.
Exclusions
Files, folders, or file types that Defender will skip during scans.
üî• Firewall & Network Protection
The Windows firewall controls what network traffic is allowed in and out through ports. It applies different rules depending on which type of network you're connected to.
Network Profiles
Domain
Active when connected to a workplace network with a domain controller. Managed by IT administrators.
Private
For trusted networks like your home. Allows device discovery and file sharing.
Public
The default for unknown networks like coffee shops and airports. Most restrictive ‚Äî blocks incoming connections.
üí° Per-profile control

Each profile can be independently toggled on or off, set to block all incoming connections, and configured to allow specific apps through. Advanced rules can be configured through wf.msc (Windows Firewall with Advanced Security).

üåê App & Browser Control
Microsoft Defender SmartScreen protects against phishing websites, malware sites, and malicious downloads.
SmartScreen Modes
Warn
The default. Alerts you about suspicious content but lets you proceed if you choose.
Block
Stops dangerous content outright with no option to continue.
Off
No SmartScreen protection at all. Not recommended.

Also includes "Check apps and files" which scans unrecognised downloads from the web, and "Exploit protection" which guards against vulnerability-based attacks.

üîê Device Security & Encryption
Hardware-level protections that go deeper than software, plus drive encryption to protect data if a device is lost or stolen.
Core Isolation / Memory Integrity
Prevents malicious code from being injected into high-security system processes. Works at the hardware virtualisation level to isolate critical parts of the OS.
Trusted Platform Module (TPM)
A physical chip on the motherboard dedicated to cryptographic operations. It's tamper-resistant and isolated from software, meaning malware cannot interfere with it. Used by BitLocker and other security features.
BitLocker Control Panel > BitLocker
Full drive encryption that scrambles all data on a drive. If a laptop is stolen, the files can't be read without the encryption key. Works best with TPM version 1.2 or later. On systems without TPM, a USB drive can store the startup key as an alternative.
üì∏ Volume Shadow Copy Service (VSS)
VSS creates point-in-time snapshots (shadow copies) of files and volumes, stored in the System Volume Information folder. This enables system restore and file recovery.
What VSS enables
Create restore points
Save the current system state before making changes
System restore
Roll back the entire system to a previous restore point
Configure restore settings
Choose which drives are protected and how much space to use
Delete restore points
Remove old snapshots to free disk space
‚ö†Ô∏è Ransomware warning

Ransomware frequently targets and deletes shadow copies to prevent victims from recovering their files. This is why off-site or offline backups are essential ‚Äî VSS alone is not a backup strategy.

üïµÔ∏è Living Off The Land

Attackers often use these same built-in Windows tools to avoid detection ‚Äî a tactic called Living Off The Land (LOtL). Because the tools are legitimate system utilities, their use doesn't trigger antivirus alerts the way malware would. Understanding what these tools do helps you spot when they're being misused.
