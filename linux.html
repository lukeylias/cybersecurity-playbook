üö© Understanding Flags
Flags are options you pass to commands to change their behavior. The dash tells the system that what follows isn't a file or argument ‚Äî it's a specific instruction for how to run the command.
ls -l -a
Two separate short flags
ls -la
Same thing ‚Äî short flags can combine
grep --ignore-case
Long flag (double dash) ‚Äî same as -i
Without the dash, the system would treat -l as a filename instead of an instruction.
