GET
Retrieve data from the server without modifying anything
Loading a webpage, fetching an image, API data retrieval
POST
Submit data to create a new resource on the server
Login forms, file uploads, creating accounts, posting comments
PUT
Update or completely replace an existing resource
Updating profile information, replacing a document
DELETE
Remove a resource from the server
Deleting a post, removing an account, clearing data
PATCH
Partially modify a resource (unlike PUT which replaces entirely)
Updating just your email without sending full profile
HEAD
Same as GET but returns only headers, no body
Checking if a file exists or its size before downloading
OPTIONS
Discover what methods a server supports
