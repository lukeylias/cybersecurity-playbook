Ports Reference

The Definitive Guide to Network Ports

What is a Port?

A port is a numbered endpoint that identifies a specific service or conversation. Think of an IP address as a building's address, and ports as the different doors inside.

Destination vs Source Ports

Port Type Who Uses It Purpose Example
Destination Port Server "Which service do you want?" 443 — you're requesting HTTPS
Source Port Client "Where should I send the reply?" 52847 — random, temporary
When your browser connects to a website, it uses a random high source port (like 52847) and connects to the server's destination port (like 443). The server responds to your source port so your device knows which application the response belongs to.
Your Laptop
192.168.1.50:52847
Source port
→
Google
142.250.80.46:443
Destination port
Your browser uses a random source port; the server listens on a well-known destination port

⚠
Security Note

Open ports are entry points. In security assessments, always scan for unexpected open ports — they can indicate misconfiguration, backdoors, or unnecessary attack surface. Services like Telnet (23), FTP (21), and unencrypted protocols should be disabled or replaced with secure alternatives.

Port Ranges

Range Name Description
0–1023 Well-known ports Reserved for common services (HTTP, SSH, etc.)
1024–49151 Registered ports Assigned to specific applications
49152–65535 Dynamic/ephemeral Used for temporary source ports

Glossary

Key terms used on this page.

TCP
Transmission Control Protocol — reliable, connection-based delivery. Ensures data arrives complete and in order.
UDP
User Datagram Protocol — fast but unreliable. No guarantee of delivery or order.
FTP
File Transfer Protocol — used to transfer files between machines.
SSH
Secure Shell — encrypted remote command-line access to a machine.
Telnet
Older remote access protocol — unencrypted, insecure, largely replaced by SSH.
SMTP
Simple Mail Transfer Protocol — used for sending emails.
DNS
Domain Name System — translates domain names (google.com) to IP addresses.
HTTP
HyperText Transfer Protocol — how browsers request and receive web pages.
HTTPS
HTTP Secure — encrypted version of HTTP using TLS.
POP3
Post Office Protocol 3 — downloads emails from server to your device.
IMAP
Internet Message Access Protocol — syncs emails with server (emails stay on server).
SMB
Server Message Block — Windows protocol for file and printer sharing.
MySQL
An open-source relational database system. Stores and retrieves structured data.
PostgreSQL
Another open-source relational database system, known for advanced features.
RDP
Remote Desktop Protocol — Windows protocol for graphical remote access to a machine.
Banner grab
Connecting to a port to see what information the service reveals about itself (version, software).
enum4linux
A tool for enumerating information from Windows/SMB systems (users, shares, etc.).
smbclient
A command-line tool for accessing SMB/Windows file shares.
rdesktop
A Linux tool for connecting to Windows machines via RDP.
xfreerdp
Another Linux tool for RDP connections, with more features than rdesktop.
