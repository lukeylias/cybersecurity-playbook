Cmdlet Reference

PowerShell is a powerful task automation and configuration management shell. Click any cmdlet below to see it in action in the terminal.

Verb-Noun Convention
PowerShell uses a Verb-Noun naming convention (e.g., Get-Process, Set-Location). Common verbs: Get, Set, New, Remove, Start, Stop, Invoke. PowerShell is not case-sensitive, but the convention is PascalCase.

üìñ Glossary
Cmdlet
A PowerShell command ‚Äî uses Verb-Noun naming convention (e.g., Get-Process)
Alias
A shortcut name for a cmdlet (e.g., ls is alias for Get-ChildItem)
Pipeline
Passing output from one cmdlet to another using the pipe symbol |
ScriptBlock
A block of code enclosed in { } that can be executed
PascalCase
Capitalising the first letter of each word (e.g., Get-Process)
-Property
A parameter used to specify which property to act on
Where-Object
Cmdlet for filtering results based on conditions
Invoke-Command
Cmdlet for running commands locally or on remote machines
