System Configuration & Administration Tools

Windows has a deep set of built-in administration tools for configuring, monitoring, and troubleshooting the system. Most can be launched from the MSConfig Tools tab or by typing their command name into Run (Win+R).

‚öôÔ∏è System Configuration (MSConfig)
MSConfig is the starting point for Windows troubleshooting. It controls what loads at boot and acts as a launcher for other admin tools. Open it with msconfig from Run.
üí° The Tools tab shortcut

MSConfig's Tools tab is a central hub that lists most Windows admin utilities in one place. Select any tool and the exact launch command appears in the "Selected command" section at the bottom. This is referenced throughout Windows admin work as the quickest way to find and open utilities.

General
Controls startup mode: Normal (loads everything), Diagnostic (basic devices and services only), or Selective (you choose what loads). Used to isolate what's causing boot problems.
Boot
Configure boot options like Safe Boot, timeout settings, and multi-OS boot order.
Services
Lists all system services with their manufacturer and status. The "Hide all Microsoft services" checkbox isolates third-party services ‚Äî useful for finding problematic software.
Startup
Redirects to Task Manager's Startup tab in modern Windows. Shows what programs launch at login and their performance impact.
Tools
Central launcher for admin utilities. Each entry shows the tool name, description, and launch command.
üîß Administration Tools
These utilities handle day-to-day system administration. Each can be found in MSConfig's Tools tab or launched directly with the command shown.
Computer Management compmgmt.msc
Central console with three sections. System Tools contains Task Scheduler (automates tasks on schedules or triggers), Event Viewer (logs system events for auditing and troubleshooting), Shared Folders (shows network shares ‚Äî hidden shares end with $), Local Users and Groups, Performance Monitor, and Device Manager. Storage contains Disk Management for drives, partitions, and drive letters. Services and Applications shows background services with their status and startup type (Automatic, Manual, or Disabled) plus WMI Control.
System Information msinfo32
Full system profile across three sections: Hardware Resources (IRQs, memory addresses), Components (display, network, storage devices), and Software Environment (running tasks, environment variables, network connections). Has a search bar at the bottom for quick lookups.
Resource Monitor resmon
Real-time performance dashboard that goes deeper than Task Manager. Separate tabs for CPU, Memory, Disk, and Network plus an Overview. Live graphs on the right show activity in real time. Used for tracking down performance issues, deadlocked processes, and file locking conflicts.
Registry Editor regedit
Edits the Windows Registry, a hierarchical database storing all system configuration: user profiles, installed apps, hardware, ports, and folder settings. Important for security because malware often hides entries here. Incorrect edits can break the system.
‚ö†Ô∏è Registry warning

The registry is powerful but dangerous. Incorrect changes can make Windows unbootable. Always back up the registry before editing, and never modify entries you don't understand.

User Account Control via MSConfig Tools tab
Security feature that prompts users when apps try to make system changes. Four notification levels from "Always notify" (highest security) down to "Never notify" (no protection). Default is the second level ‚Äî notify when apps make changes, but not for user-initiated settings changes.
