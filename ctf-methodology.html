Service Tools What to look for

Quick Troubleshooting

Problem Try This
Nmap slow Use -F for fast scan, or -p <specific ports>
Can't reach target Check VPN connection, verify IP
Gobuster finds nothing Try different wordlist, check for redirects
Exploit doesn't work Check versions match, try alternatives
Shell dies immediately Use a more stable shell, try different payload
